# 🧬 Стохастичний клітинний автомат для моделювання росту пухлини

Python-реалізація **стохастичного клітинного автомату (CA)**, що моделює динаміку росту аваскулярної пухлини — включаючи гетерогенну популяцію клітин (звичайні та стовбурові ракові клітини), імунну відповідь організму, формування некротичного ядра та терапевтичні втручання (хіміотерапія, радіотерапія, імунотерапія).

Модель побудована на основі методологій з двох рецензованих наукових статей (див. [Джерела](#джерела)).

## Огляд

Проєкт реалізує **двовимірний стохастичний клітинний автомат**, що моделює ріст пухлини на клітинному рівні. Починаючи з однієї ракової клітини на дискретній решітці, симуляція відстежує проліферацію, міграцію, апоптоз, взаємодію з імунними клітинами та накопичення некротичної тканини.

Клітинний автомат визначається як четвірка **CA = (C, n, S, f)**:

| Символ | Назва | Опис |
|--------|-------|------|
| **C** | Простір клітин | 2D решітка розміром N×N |
| **n** | Сусідство | Сусідство Мура — 8 навколишніх клітин |
| **S** | Множина станів | {EMPTY, HEALTHY, CANCER, CANCER_STEM, IMMUNE, NECROTIC} |
| **f** | Функція переходу | Стохастичні ймовірнісні правила оновлення станів |

## Стани клітин

| Код | Символ | Стан | Опис |
|-----|--------|------|------|
| 0 | `·` | EMPTY | Порожня позиція |
| 1 | `○` | HEALTHY | Нормальна клітина тканини |
| 2 | `●` | CANCER (RTC) | Звичайна ракова клітина — обмежений потенціал поділу |
| 3 | `★` | CANCER_STEM (STC) | Стовбурова ракова клітина — безсмертна |
| 4 | `◆` | IMMUNE | Імунна клітина — знищує ракові клітини |
| 5 | `✕` | NECROTIC | Мертва клітина — з часом розпадається |

## Сценарії симуляції

| Сценарій | Опис | Очікуваний результат |
|----------|------|----------------------|
| `rtc` | Тільки звичайні ракові клітини | Пухлина зникає — RTC вичерпують потенціал поділу |
| `stem` | Стовбурова STC без імунної системи | Стабільна пухлина — STC безперервно поповнює RTC |
| `immune` | STC + здорова тканина + імунна система | Імунна система сповільнює, але не зупиняє ріст |
| `multi` | Кілька пухлин по всій решітці | Взаємодія та злиття пухлин |
| `no-treatment` | STC + імунна, без терапії | Базовий сценарій для порівняння |
| `chemo` | Хіміотерапія кожні N кроків | Масовий некроз RTC, STC виживають |
| `radio` | Радіотерапія після симуляції | Кругла зона некрозу в епіцентрі |

## Терапевтичні методи

### Хіміотерапія (`chemotherapy.py`)
Моделює системний вплив цитотоксичних препаратів. Застосовується глобально до всієї решітки кожні `--chemo-interval` кроків.

- Висока ефективність проти RTC (`p_kill_rtc = 0.8`)
- Низька ефективність проти STC (`p_kill_stc = 0.3`) — стовбурові клітини резистентні
- Токсичність для здорових клітин (`p_tox_healthy = 0.05`)

**Очікуваний результат:** ефект "швейцарського сиру" — маса пухлини перетворюється на некроз, але STC виживають у вигляді розсіяних острівців. Рецидив неминучий.

### Радіотерапія (`radiotherapy.py`)
Моделює спрямований промінь іонізуючого випромінювання. Застосовується один раз після симуляції в центр решітки.

Використовується Евклідова відстань від епіцентру:
$$d = \sqrt{(x - x_0)^2 + (y - y_0)^2}$$

Усі клітини в радіусі `--radio-radius` отримують критичне ушкодження.

**Очікуваний результат:** ідеальна кругла зона некрозу. Якщо пухлина вийшла за межі радіусу — краї продовжують рости.

### Імунотерапія (`immunotherapy.py`)
Моделює таргетну терапію (CAR-T клітини). Діє у два етапи:
1. **Boost:** наявні IC розширюють зону пошуку до радіусу `boost_radius`
2. **Рекрутинг:** нові IC з'являються на порожніх і некротичних клітинках

**Очікуваний результат:** пухлина деградує з країв, IC скупчуються навколо залишків пухлини.

## Параметри моделі

| Параметр | Значення | Опис |
|----------|----------|------|
| `PP` | 4.17% | Ймовірність проліферації (`Δt / CCT`) |
| `PA` | 0.5% | Ймовірність апоптозу (тільки RTC) |
| `Pm` | 2% | Ймовірність міграції |
| `Ps` | 1% | Симетричний поділ STC |
| `pmax` | 10 | Максимальний потенціал поділу RTC |
| `pdT` | 15% | Ймовірність що IC вб'є ракову клітину |
| `pdI` | 5% | Ймовірність що ракова клітина вб'є IC |

## Як запустити

### Вимоги
```bash
pip install numpy matplotlib
```

### Запуск сценаріїв через `main.py`

```bash
# Без лікування
python main.py --scenario no-treatment --steps 200 --grid-size 80

# Тільки RTC (пухлина зникає)
python main.py --scenario rtc --steps 200 --grid-size 80

# Стовбурова пухлина
python main.py --scenario stem --steps 200 --grid-size 80

# З імунною системою
python main.py --scenario immune --steps 200 --grid-size 80

# Кілька пухлин
python main.py --scenario multi --steps 200 --grid-size 120

# Хіміотерапія кожні 10 кроків
python main.py --scenario chemo --steps 200 --chemo-interval 10

# Радіотерапія з радіусом 8
python main.py --scenario radio --steps 200 --radio-radius 8

# З фіксованим seed для відтворюваності
python main.py --scenario chemo --steps 200 --seed 42
```

### Всі аргументи

| Аргумент | За замовчуванням | Опис |
|----------|------------------|------|
| `--steps` | 200 | Кількість кроків симуляції |
| `--grid-size` | 80 | Розмір решітки N×N |
| `--scenario` | no-treatment | Сценарій запуску |
| `--chemo-interval` | 10 | Кожні скільки кроків застосовувати хімію |
| `--radio-radius` | 6 | Радіус дії радіотерапії |
| `--seed` | None | Random seed для відтворюваності |

### Генерація графіків

```bash
python plot_simulation.py
# або з параметрами:
python plot_simulation.py 70 150
```

### Тест усіх методів терапії

```bash
python test_therapy.py
```
