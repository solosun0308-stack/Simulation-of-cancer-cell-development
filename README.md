# Моделювання терапевтичних стратегій для стохастичного клітинного автомату пухлини

Python-реалізація комплексу терапевтичних втручань (хіміотерапія, радіотерапія, імунотерапія) для стохастичної CA-моделі аваскулярної пухлини.

Цей модуль розширює базове ядро симуляції (`tumor_new.py`), додаючи зовнішні збурення до функції переходу станів $f$. Моделі терапії враховують просторову гетерогенність пухлини, наявність резистентних стовбурових ракових клітин (STC) та взаємодію з мікрооточенням. Результати демонструють різні патерни некрозу та виживання клітин залежно від обраного фізичного або біологічного методу впливу.

---

## Біологічне та математичне обґрунтування методів

У межах клітинного автомату $CA = (C, n, S, f)$ терапія розглядається як примусова зміна стану клітини на $NECROTIC$ ($5$) або $EMPTY$ ($0$) на основі глобальних або локальних просторових правил.

### 1. Хіміотерапія (Системний стохастичний вплив)

Моделює введення цитотоксичних препаратів у кровотік. Оскільки модель аваскулярна, припускається рівномірна дифузія препарату по всій решітці (незалежно від просторових координат).

- **Механізм:** Глобальний стохастичний перехід. Кожна клітина $RTC$ або $STC$ має незалежну ймовірність загинути.
- **Біологічна реалістичність:** Модель враховує **резистентність стовбурових клітин**. Ймовірність знищення звичайної ракової клітини ($p_{kill\_rtc}$) значно вища за ймовірність знищення стовбурової ($p_{kill\_stc}$). Також враховується загальна токсичність для здорових тканин.

### 2. Радіотерапія (Локалізований геометричний вплив)

Моделює спрямований промінь іонізуючого випромінювання. На відміну від хіміотерапії, цей метод є суворо просторово залежним.

- **Механізм:** Використовується Евклідова відстань $d$ від заданого епіцентру $(x_0, y_0)$ до клітини $(x, y)$:

$$d = \sqrt{(x - x_0)^2 + (y - y_0)^2}$$

Якщо $d \le R$ (радіус променя), клітина отримує критичне ушкодження ДНК і переходить у стан $NECROTIC$ з імовірністю, близькою до 1.

- **Емерджентність:** Здорові клітини, що потрапили в радіус дії, також гинуть, формуючи чітку зону некротичного ураження (шрам).

### 3. Імунотерапія (Модифікація локального сусідства)

Моделює таргетну терапію (наприклад, CAR-T клітини або інгібітори контрольних точок). Це не пряме знищення раку, а «програмування» імунної системи моделі.

- **Механізм:** Діє у два етапи:
  1. **Просторовий Boost:** Наявні клітини $IMMUNE$ розширюють зону пошуку цілі за межі стандартного сусідства Мура (до радіуса $R_{boost}$) і агресивно знищують пухлину.
  2. **Рекрутинг:** Стохастична поява нових лімфоцитів ($IMMUNE$) на порожніх або некротичних клітинках, що імітує масовану імунну відповідь через капіляри.

##  Параметри терапевтичних моделей

Усі методи лікування параметризовані для можливості тонкого налаштування ефективності та токсичності:

| Параметр | Модуль | Значення | Опис |
|:---|:---|:---|:---|
| `p_kill_rtc` | Хімія / Радіо | 0.80 - 0.99 | Ймовірність знищення звичайної клоногенної клітини (RTC) |
| `p_kill_stc` | Хімія / Радіо | 0.30 - 0.70 | Ймовірність знищення безсмертної стовбурової клітини (STC) |
| `target_r, target_c` | Радіо | $N/2, N/2$ | Координати епіцентру спрямованого променя |
| `radius` | Радіо | $R_{beam}$ | Радіус дії іонізуючого випромінювання |
| `boost_radius` | Імуно | 2-3 клітини | Розширене сусідство для активованих Т-кілерів |
| `recruitment_rate` | Імуно | 0.05 | Швидкість генерації нових імунних клітин з кровотоку |

##  Структура терапевтичних модулів

Архітектура побудована за принципом *Separation of Concerns* (Розділення відповідальності), де кожен вид терапії інкапсульований в окремий файл:

- `chemotherapy.py` — модуль дифузної фармакологічної дії.
- `radiotherapy.py` — модуль просторової променевої терапії.
- `immunotherapy.py` — модуль клітинної імунної відповіді.
- `test_therapy.py` — **Інтеграційний скрипт**. Виконує роль контролера:
  1. Отримує згенеровану пухлину з базового ядра `TumorCA`.
  2. Створює незалежні глибокі копії масиву (`numpy.copy()`) для чистоти експерименту.
  3. Застосовує кожен метод на ідентичній пухлині для коректного порівняльного аналізу.
  4. Відмальовує результати через нативний метод ядра `to_ascii()`.

Щоб запустити симуляцію росту з подальшим тестуванням усіх методів терапії:

```bash
python test_therapy.py
```

### Емерджентна поведінка при лікуванні (Очікувані результати)

1. **Після хіміотерапії:** Спостерігається ефект "швейцарського сиру". Маса пухлини перетворюється на некроз (`✕`), але резистентні STC (`★`) виживають у вигляді розсіяних "острівців". Це математично доводить неминучість рецидиву після класичної хімії.
2. **Після радіотерапії:** Формується ідеальна кругла зона випаленого некрозу незалежно від типу клітин. Проте, якщо пухлина асиметрична і вийшла за межі `radius`, краї пухлини продовжуватимуть ріст.
3. **Після імунотерапії:** Пухлина деградує з країв (фронт атаки лімфоцитів). Спостерігається масове скупчення значків `◆` навколо залишків пухлини, що імітує запальний процес та локалізацію антигенів.


*Примітка: Усі модулі використовують систему станів (EMPTY, HEALTHY, CANCER, CANCER_STEM, IMMUNE, NECROTIC) та їх ASCII-візуалізацію, що повністю синхронізована з ядром `tumor_new.py`.*
